# Incident Response Playbook (Notion)

## Escalation Policy
- Page primary on-call within 5 minutes.
- Escalate to secondary if no response in 10 minutes.

## Communication
- Post updates in #incidents every 15 minutes.
- Create a timeline doc for postmortem.

## Closure Checklist
- Confirm monitoring is green.
- Capture root cause and remediation items.
